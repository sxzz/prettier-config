# @sxzz/prettier-config

[![npm version][npm-version-src]][npm-version-href]
[![npm downloads][npm-downloads-src]][npm-downloads-href]

Prettier config.

## Install

```bash
npm i -D @sxzz/prettier-config
```

## Features

- 2 spaces
- No semicolons
- Single quotes
- Trailing commas
- Ignore common files (`dist`, `pnpm-lock.yaml`...)
